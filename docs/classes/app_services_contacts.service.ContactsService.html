    Class ContactsService

    Service for managing contacts in the Firestore database. Provides methods to add, delete, update, and retrieve contacts. Additionally, it allows for subscribing to changes in the contacts list.

    Remarks

    This service interacts with Firestore to fetch, update, and delete contact data. It also manages the contact list and ensures the user can only edit their own contacts.
